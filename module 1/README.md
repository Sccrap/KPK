# Автоматизация настройки сети — ALT Linux (Модуль 1)

## Структура скриптов

| Скрипт | Устройство | Описание |
|--------|-----------|----------|
| `01_isp.sh` | ISP | Имя хоста, IP-адреса, NAT, forwarding |
| `02_hq-rtr.sh` | HQ-RTR | Имя хоста, IP (ens19), forwarding, NAT, VLAN (OVS), GRE-туннель, OSPF, DHCP |
| `03_br-rtr.sh` | BR-RTR | Имя хоста, IP-адреса, forwarding, NAT, GRE-туннель, OSPF |
| `04_hq-srv.sh` | HQ-SRV | Имя хоста, IP, пользователи, SSH, DNS (BIND), часовой пояс |
| `05_hq-cli.sh` | HQ-CLI | Имя хоста, DHCP-клиент, часовой пояс |
| `06_hq-sw.sh` | HQ-SW | Имя хоста, IP, часовой пояс |
| `07_br-srv.sh` | BR-SRV | Имя хоста, IP, пользователи, SSH, часовой пояс |

## Порядок выполнения

1. Скопировать нужный скрипт на соответствующее устройство
2. Дать права на выполнение: `chmod +x <script>.sh`
3. Запустить от root: `bash <script>.sh`

## Важно

- Перед запуском проверьте имена интерфейсов (`ip -c a`) и при необходимости измените переменные в начале каждого скрипта.
- Домен: `au-team.irpo`
- Пароли по умолчанию: `P@ssw0rd` (серверы), `P@$$word` (маршрутизаторы)
- После настройки GRE/OSPF на обоих роутерах может потребоваться перезагрузка.
